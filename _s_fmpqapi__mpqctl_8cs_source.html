12 // All functions below are actual functions that are part of this
13 // library and do not need any additional dll files. It does not
14 // even require Storm to be able to decompress or compress files.

16 // This library emulates the interface of Lmpqapi and Storm MPQ
17 // functions, so it may be used as a replacement for them in
18 // MPQ extractors/archivers without even needing to recompile
19 // the program that uses Lmpqapi or Storm. It has a few features
20 // not included in Lmpqapi and Storm, such as extra flags for some
21 // functions, setting the locale ID of existing files, and adding
22 // files without having to write them somewhere else first. Also,
23 // MPQ handles used by functions prefixed with "SFile" and "Mpq"
24 // can be used interchangably; all functions use the same type
25 // of MPQ handles. You cannot, however, use handles from this
26 // library with storm or lmpqapi or vice-versa. Doing so will
27 // most likely result in a crash.
28
