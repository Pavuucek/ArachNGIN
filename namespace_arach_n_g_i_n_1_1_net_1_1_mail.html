enum ArachNGIN.Net.Mail.Pop3State
Hodnoty výčtu
Unknown 

Undefined.

Authorization 

Connected to Pop3 server, but not authorized. May issue any of the following commands QUIT, USER, PASS

Transaction 

Authorized to Pop3 server, can issue any of the following commands; STAT, LIST, RETR, DELE, RSET

Update 

Quit command was sent to server indicating deleted messages should be removed.
