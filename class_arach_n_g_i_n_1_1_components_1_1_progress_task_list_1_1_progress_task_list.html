Detailed Description

Implements a task-list control, derived from Panel. It displays a vertical list of tasks, with an arrow beside the current task. When the task is complete, a green 'tick' icon is placed next to it to indicate that is is finished. To use this control, you must call Start() to begin, and NextTask() to advance to the next task.
