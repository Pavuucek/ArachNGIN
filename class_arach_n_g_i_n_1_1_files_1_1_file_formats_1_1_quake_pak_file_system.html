bool ArachNGIN.Files.FileFormats.QuakePakFileSystem.AskFile ( string  sFile)

Asks for a file and places it in a temp directory

  • if it already is in temp dir it does nothing
  • if it exists in program dir it gets copied to temp dir
  • if it exists in one of PAK files it gets extracted to temp dir In this order.
